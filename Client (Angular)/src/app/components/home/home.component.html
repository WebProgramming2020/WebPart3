JWT Tokens

Fully featured authentication using JSON web tokens. Register and then Login and store user data

To Register =>

Enter the following:

Name: Anything you like

Username: Anything you like, as long as it is not currently in use

Email: Any Valid Email you like, as long as it is not already associated with an account

Password: Must be at least 8 characters, have one special character, and one digit

To Login =>

Enter the following

Username:The Username you chose when you registered

Password:The Password you chose when you registered

**HINT**:A working example password for testing this is "iloveangular11!". Copy and paste if you like!

(Don't forget the Exclamation Mark!))

Auth Guards

Links to certain routes are invisible to the logged out user

Try to access these routes without logging in (through the browser search bar) and you will be redirected to the login page
